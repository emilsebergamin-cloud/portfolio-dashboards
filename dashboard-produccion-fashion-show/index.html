Sobre este proyecto
Qué es: Dashboard de producción para organizar un desfile de moda. Centraliza info del show (venue, fecha, looks), colección con estados por pieza, timeline de producción, contactos del equipo, presupuesto con categorías y estados, moodboard visual, links de recursos, notas y dirección creativa — todo editable en tiempo real.
Lógica: La producción de un desfile involucra decenas de proveedores, decisiones de presupuesto, timelines ajustados y mucha coordinación. La información suele vivir dispersa entre emails, planillas, WhatsApp y carpetas de Drive. Este dashboard unifica todo en una sola interfaz donde la diseñadora y su equipo pueden ver el estado completo del proyecto sin pedir updates a nadie.
Stack: HTML + CSS + JavaScript vanilla. Single-file. En esta versión demo los datos se guardan en localStorage. La versión en producción utiliza Firebase Firestore para sincronización en tiempo real entre dispositivos + Firebase Storage para imágenes del moodboard. Export a JSON para backup. Responsive. Logo SVG inline. Deploy en Vercel.
Rol: Product management, diseño de UX/UI, desarrollo front-end, integración con Firebase (Firestore + Storage), diseño del sistema de presupuesto con categorías, deploy y entrega al equipo de producción.
⚠ Nota: Toda la información es ficticia. Nombres, presupuestos, contactos y datos del evento fueron creados para esta demostración. El dashboard original fue desarrollado para una marca real cuya identidad se preserva por confidencialidad.
