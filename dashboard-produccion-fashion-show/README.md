# Dashboard — Producción Fashion Show

Dashboard construido para coordinar la producción de un desfile de moda. Pensado como hub de información para el equipo: modelos, vestuario, backstage, sponsors, timing y logística del evento.

## Contexto

Proyecto **freelance**. La versión publicada acá es una demo **anonimizada** — la marca, los nombres del equipo, la ubicación del desfile y los sponsors fueron reemplazados para proteger la privacidad del cliente original.

## Problema que resolvía

Una producción de fashion show involucra decenas de personas en paralelo: director creativo, producción, modelos, vestuaristas, estilistas, fotografía, prensa, sponsors. La información crítica (orden de salidas, looks por modelo, timing del backstage) suele vivir en planillas que se actualizan mal y en chats donde todo se pierde.

El dashboard centraliza todo: cada modelo tiene su ficha con su look asignado, el timing está visible para todos los departamentos, los sponsors tienen su sección dedicada. Un solo link, siempre actualizado.

## Decisiones de diseño

- **Estética editorial minimalista**: la producción de moda demanda diseño que respete el mundo que cubre.
- **Tipografía sans-serif de alta legibilidad**: se lee rápido en backstage con poca luz.
- **Layout tipo "programa de mano"**: familiar para equipos de moda, no se siente "tech".
- **Jerarquía clara**: lo que importa en el momento del show está arriba, lo secundario abajo.

## Stack

- HTML + CSS + JavaScript vanilla, un solo archivo.

## Cómo verlo

Abrir `index.html` en cualquier navegador.
