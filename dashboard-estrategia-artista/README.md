# Dashboard — Estrategia de Artista

Dashboard construido para planificación estratégica anual de una artista emergente. Pensado como plan de ruta para el año: pilares de marca, referencias visuales, cronograma de lanzamientos, presupuestos y prioridades.

## Contexto

Proyecto realizado para **XIX Consultoría de Artistas**. La versión publicada acá es una demo **anonimizada** — la artista real, los datos de redes, los números de presupuesto y los links fueron reemplazados para proteger la privacidad del proyecto original.

## Problema que resolvía

Cuando arranca un año nuevo con una artista, el plan estratégico suele terminar en documentos que nadie vuelve a abrir. El management lo arma, se presenta en una reunión, y después se olvida. La intención se pierde en la ejecución.

El dashboard transforma el plan estratégico en un espacio vivo: cada pilar tiene su sección, cada acción tiene su calculadora de presupuesto, cada referencia está linkeada. Se vuelve la página que el management abre cuando tiene dudas.

## Decisiones de diseño

- **Tema dark con violeta y sky blue**: cercano al universo estético de la artista.
- **Calculadora de presupuestos embebida**: para evaluar escenarios de producción sin salir del dashboard.
- **Tipografía editorial (Playfair Display + DM Sans)**: eleva el plan estratégico a un objeto visualmente cuidado.
- **Pilares como cards**: cada eje estratégico tiene su espacio visual, no se mezcla con los otros.

## Stack

- HTML + CSS + JavaScript vanilla, un solo archivo.
- Cálculos embebidos con JS para las calculadoras de presupuesto.

## Cómo verlo

Abrir `index.html` en cualquier navegador.
