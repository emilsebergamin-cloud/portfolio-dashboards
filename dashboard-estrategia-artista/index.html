Sobre este proyecto
Qué es: Plan estratégico de 3 meses para artistas musicales pop. Incluye calculadora de presupuesto interactiva con planes Bronze/Silver/Gold para media buying y producción, scheduling semanal por mes, referencias visuales y artísticas, y sección de compositores/productores con comparación de opciones.
Lógica: La planificación de un lanzamiento musical suele manejarse con tablas de Excel y cadenas de emails. Este dashboard unifica presupuesto, calendario editorial y referencias en una interfaz interactiva donde el artista y su equipo pueden tomar decisiones en tiempo real — seleccionando planes de inversión y viendo el impacto inmediato en el presupuesto total.
Stack: HTML + CSS + JavaScript vanilla. Single-file. Diseño dark theme con sidebar sticky. Calculadora de presupuesto con lógica de selección exclusiva por grupo y total acumulativo en tiempo real. Responsive con breakpoints para mobile. Deploy en Vercel.
Rol: Product management, diseño de UX/UI, desarrollo front-end, diseño del sistema de calculadora interactiva, estrategia de contenido musical, planificación del calendario editorial, deploy y entrega al equipo de consultoría.
⚠ Nota: Toda la información es ficticia. Nombres, métricas, presupuestos y referencias fueron creados para esta demostración. El dashboard original fue desarrollado para un cliente real cuya identidad se preserva por confidencialidad.
