Sobre este proyecto
Qué es: Dashboard de gestión de contenido para marcas de wellness y cuidado personal. Permite organizar contenido por tipo (carrusel, reel, story, post), gestionar un moodboard visual, y mantener un pipeline de ideas con flujo kanban (ideas → en proceso → publicado). Incluye exportación a PDF por sección para compartir con content managers o creadoras de contenido.
Lógica: Cuando una marca pequeña gestiona contenido para redes, la información suele vivir dispersa entre notas del celular, Canva, Google Docs y conversaciones de WhatsApp. Este dashboard centraliza todo en un solo lugar: qué se publica, cuándo, en qué formato, qué ideas hay pendientes y qué referencias visuales definen la estética. El export a PDF permite entregar briefs profesionales sin armar un documento desde cero.
Stack: HTML + CSS + JavaScript vanilla. Single-file. En esta versión demo los datos se guardan en localStorage. La versión en producción utiliza Firebase Firestore para sincronización en tiempo real entre dispositivos (compu y celular). Export a PDF con html2pdf.js. Saludo dinámico según hora del día. Frase diaria rotativa (30 frases sobre oficio y disciplina). Deploy en Vercel.
Rol: Product management, diseño de UX/UI, desarrollo front-end, integración con Firebase, diseño del sistema de export a PDF, estrategia de contenido (incluyendo el copy del carrusel educativo precargado como ejemplo), deploy y entrega.
⚠ Nota: Toda la información presentada en este dashboard es ficticia. Los contenidos, ideas y referencias fueron creados para esta demostración. El dashboard original fue desarrollado para una clienta real cuya identidad y datos se preservan por confidencialidad.
